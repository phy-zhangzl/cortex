# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a Vulnerability

Please do not open a public issue for sensitive security problems.

Instead, report the issue privately through GitHub security reporting if enabled, or contact the maintainer directly.

Please include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Suggested mitigation if available

We will acknowledge the report as quickly as possible and work on a fix before public disclosure when appropriate.
