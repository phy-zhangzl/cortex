# Cortex App

This directory contains the Tauri application for Cortex.

## Commands

```bash
npm ci
npm run tauri:dev
npm run check
npm run tauri:build
```

## Notes

- Frontend: React + TypeScript + Vite
- Backend: Rust + Tauri
- Packaging: macOS app bundle and DMG via Tauri
